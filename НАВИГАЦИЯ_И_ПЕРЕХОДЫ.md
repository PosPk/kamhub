## 🌐 ГЛОБАЛЬНАЯ НАВИГАЦИЯ

### Header (на всех страницах)

```
┌─────────────────────────────────────────────────────────┐
│  🟨 Logo  "Kamchatour Hub"    🌓  [Войти] [CRM] [Витрина] │
└─────────────────────────────────────────────────────────┘
```

**Элементы Header:**

1. **🟨 Logo + "Kamchatour Hub"**
   - Клик → `/` (главная страница)
   - Всегда видим

2. **🌓 Theme Toggle**
   - Переключатель темы (светлая/тёмная)
   - Сохраняется в localStorage
   - Не перезагружает страницу

3. **[Войти]** → `/auth/login`
   - Для неавторизованных пользователей
   - После входа → редирект по роли

4. **[CRM]** → `/hub/operator`
   - Прямой доступ к CRM системе
   - Для операторов и партнёров

5. **[Витрина]** → `/tg` (не реализовано)
   - Telegram Mini App витрина
   - В планах

```
                    🌐 Интернет
                         ↓
                    ┌─────────┐
                    │    /    │  Главная страница

        ┌────────────────┼────────────────┐
        ↓                ↓                ↓

```

```
├── Успех → редирект по роли:
│   ├── tourist → /hub/tourist

│   ├── driver → /hub/transfer-operator
│   └── hotel_manager → /hub/stay
```

## 📄 ДЕТАЛЬНОЕ ОПИСАНИЕ ПЕРЕХОДОВ

### 🏠 ГЛАВНАЯ СТРАНИЦА (/)

**Откуда попадаем:**
- Прямой URL: `http://5.129.248.224/`
- Клик на Logo в Header (с любой страницы)
- После logout (планируется)

**Куда можем перейти:**

#### 1. Hero секция (верхняя часть)
```
┌─────────────────────────────────────────┐
│  Экосистема туризма Камчатки            │
│  [Поиск: вулканы, океан, медведи...]    │
│  [🚀 Демо]  [Войти]  [Регистрация]      │
└─────────────────────────────────────────┘
```

- **[🚀 Демо]** → `/demo` - демонстрация функций
- **[Войти]** → `/auth/login` - вход в систему
- **[Регистрация]** → `/auth/register` - выбор типа регистрации
- **Поле поиска** → пока не работает (планируется)

#### 2. Карточки ролей (8 штук)
```
┌──────────┬──────────┬──────────┬──────────┐
│ Турист   │ Оператор │   Гид    │ Трансфер │
├──────────┼──────────┼──────────┼──────────┤
│Размещение│ Сувениры │ Снаряж.  │   Авто   │
└──────────┴──────────┴──────────┴──────────┘
```

- **Турист** → `/hub/tourist`
- **Туроператор** → `/hub/tour-operator`
- **Гид** → `/hub/guide`
- **Трансфер** → `/hub/transfer-operator`
- **Размещение** → `/hub/stay`
- **Сувениры** → `/hub/souvenirs`
- **Прокат снаряжения** → `/hub/gear`
- **Прокат авто** → `/hub/cars`

#### 3. Популярные туры
- **[Тур]** → детальная страница тура (в разработке)
- Пока показываются skeleton loaders

#### 4. SOS и безопасность
```
┌─────────────────────────────────┐
│  [SOS]  [МЧС]  [Сейсмика]       │
│  [Карта Камчатки] (кликабельна) │
└─────────────────────────────────┘
```

- **[Карта Камчатки]** → `/hub/safety`

#### 5. AI-Гид
- Встроенный виджет
- Предзаполненные вопросы (кликабельны)
- Текстовое поле для вопросов
- **[Отправить]** → запрос к AI API

#### 6. Быстрые переходы (футер)
- **Каталог туров** → `/partners` (не реализовано)
- **Поиск** → `/search` (не реализовано)
- **Витрина Commerce** → `/premium` (не реализовано)
- **Витрина Adventure** → `/premium2` (не реализовано)
- **Размещение** → `/hub/stay`
- **Безопасность** → `/hub/safety`
- **Рефералы и бусты** → `/hub/operator`


## 🔄 РЕДИРЕКТЫ ПОСЛЕ АВТОРИЗАЦИИ

### Логика редиректов

```javascript
// После успешного входа
function redirectAfterLogin(user) {
  const primaryRole = user.roles[0]; // Первая роль
  
  switch(primaryRole) {
    case 'tourist':
      return '/hub/tourist';
    
    case 'operator':
      return '/hub/tour-operator';
    
    case 'guide':
      return '/hub/guide';
    
    case 'driver':
      return '/hub/transfer-operator';
    
    case 'hotel_manager':
      return '/hub/stay';
    
    case 'provider':
      return '/hub/gear';
    
    case 'restaurant_owner':
      return '/hub/souvenirs';
    
    default:
      return '/hub/tourist'; // По умолчанию
  }
}
```

| Роль | Главный хаб | Доступ к CRM | Дополнительно |

| **restaurant_owner** | `/hub/souvenirs` | ✅ Частично | Магазин, продажи |

### Мульти-роли

Если у пользователя несколько ролей:
```javascript
user = {
  roles: ['operator', 'guide', 'driver']
}
```

**Навигация:**
- Главный хаб определяется по первой роли: `/hub/tour-operator`
- Доступ ко всем хабам своих ролей через меню
- В CRM видны все связанные разделы

**Меню для мульти-роли:**
```
┌─────────────────────────────────┐
│  Мои роли:                      │
│  • Туроператор → /hub/tour-operator
│  • Гид → /hub/guide              │
│  • Водитель → /hub/transfer-operator
│  • CRM → /hub/operator           │
└─────────────────────────────────┘
```
