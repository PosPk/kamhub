## 🎭 Эффекты и анимации

### Градиенты
```css
.animate-gradient {
  background-size: 200% 200%;
  animation: gradient 15s ease infinite;
}
```

### Floating particles
```css
.animate-float {
  animation: float 6s ease-in-out infinite;
}

.animate-float-delayed {
  animation: floatDelayed 8s ease-in-out infinite;
}
```

### Scroll indicator
```css
.animate-scroll {
  animation: scroll 2s ease-in-out infinite;
}
```

### Hover эффекты
- `hover:scale-105` - увеличение при наведении
- `hover:-translate-y-4` - поднятие карточки
- `hover:rotate-3` - лёгкий поворот иконки
- `hover:gap-4` - расширение расстояния для стрелок
- `hover:bg-white/20` - изменение прозрачности

## 📐 Технические детали

### Убраны ограничения
```diff
- <div className="max-w-7xl mx-auto px-4 sm:px-6">
+ <div className="w-full px-6 md:px-12 lg:px-20">
```

### Адаптивные отступы
- `px-6` - на мобильных (24px)
- `md:px-12` - на планшетах (48px)
- `lg:px-20` - на десктопах (80px)

### Rounded corners
- Hero cards: `rounded-[3rem]` (48px!)
- Role cards: `rounded-[2.5rem]` (40px)
- Icon boxes: `rounded-3xl` (24px)

### Glassmorphism
```css
bg-white/10 backdrop-blur-2xl border border-white/20
```

## 📱 Адаптивность

### Мобильные (< 768px)
- `text-8xl` → время крупно
- `grid-cols-1` → карточки в колонку
- `px-6` → отступы 24px

### Планшеты (768px - 1024px)
- `md:text-9xl` → время ещё крупнее
- `md:grid-cols-2` → карточки по 2 в ряд
- `md:px-12` → отступы 48px

### Десктоп (> 1024px)
- `lg:text-9xl` → максимальный размер
- `lg:grid-cols-3` → карточки по 3 в ряд
- `lg:px-20` → отступы 80px (но контент всё равно на полную ширину!)
