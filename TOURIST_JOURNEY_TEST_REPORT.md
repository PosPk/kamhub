#### ✅ Исправление #2: Middleware userId header
**Файл:** `middleware.ts`  
**Проблема:** API endpoints возвращают 401 "пользователь не авторизован"

**Решение:**
- Добавлена декодирование JWT токена
- Добавлена передача userId в x-user-id header
- Добавлена обработка ошибок и логирование

```typescript
// Декодирование JWT
const verified = await jwtVerify(token, secret);
const userId = verified.payload.sub || verified.payload.userId;

// Передача userId в headers
requestHeaders.set('x-user-id', String(userId));
```

**Статус:** ✅ ИСПРАВЛЕНО
